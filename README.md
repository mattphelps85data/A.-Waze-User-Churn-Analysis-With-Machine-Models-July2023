# WAZE 
## User Churn Analysis and Prediction with Machine Models

## Description
This is an analysis of churn based on a Waze provided dataset.  EDA, Regression Models, and Machine Models were used to gather insights into user churn for Waze.

## Tech

**Language:** Python

**IDE:** Spyder, JupyterNotebook

**Libraries:** pandas, numpy, matplotlib, seaborn, scipy, sklearn, xgboost

**Visualization Tool:** Tableau
